## 使用方式：
1. 安装脚本后, 推特**列表**的查看成员界面多出两个按钮。
2. 用电脑打开列表 [点击跳转到列表地址](https://twitter.com/i/lists/1677334530754248706)，
3. 点列表**封面图下方**的查看成员（members），打开列表成员弹框, 在弹框的右上角有"全部屏蔽"按钮
